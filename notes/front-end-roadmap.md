Lộ trình học Front-end (20 giờ/tuần)Giai đoạn 1: Xây dựng khung xương và lớp áo (Tuần 1 - Tuần 4)Mục tiêu: Thành thạo cấu trúc trang web và giao diện Responsive (đẹp trên cả điện thoại và máy tính).Nội dung học:Tuần 1: HTML5 (Các thẻ nội dung, thẻ form, cấu trúc chuẩn SEO).Tuần 2: CSS3 cơ bản (Màu sắc, font chữ, mô hình Box Model, Selector).Tuần 3: CSS nâng cao (Flexbox, CSS Grid) để dàn trang.Tuần 4: Responsive Web Design (Media Queries) và làm quen với hệ thống quản lý mã nguồn Git / GitHub.Thực hành: Dựng lại giao diện của các trang web như Instagram (giao diện máy tính), trang chủ Shopee hoặc một trang Landing Page bán hàng đơn giản từ file thiết kế.Giai đoạn 2: Tạo "linh hồn" cho Website với JavaScript (Tuần 5 - Tuần 12)Mục tiêu: Giúp trang web có thể tương tác (bấm nút mở menu, gửi dữ liệu, làm hiệu ứng trượt).Nội dung học:Tuần 5 - 6: Cú pháp cơ bản (Biến, Kiểu dữ liệu, Câu lệnh điều kiện if/else, Vòng lặp for/while, Hàm).Tuần 7 - 8: DOM Manipulation (Cách JavaScript can thiệp, thay đổi HTML và CSS khi người dùng click, cuộn chuột...).Tuần 9 - 10: JavaScript nâng cao (Bất đồng bộ - Asynchronous, Fetch API/Axios để lấy dữ liệu từ Server về hiển thị lên giao diện).Tuần 11 - 12: Thực hành dự án JavaScript thuần.Thực hành: Làm ứng dụng Music Player (nghe nhạc), App thời tiết (lấy dữ liệu thực tế qua API), hoặc ứng dụng quản lý công việc (Todo List).Giai đoạn 3: Tăng tốc với Framework - ReactJS (Tuần 13 - Tuần 20)Mục tiêu: Học công cụ giúp xây dựng các website lớn, phức tạp một cách nhanh chóng (ReactJS hiện có nhu cầu tuyển dụng cao nhất tại Việt Nam).Nội dung học:Tuần 13 - 14: Khái niệm Component, JSX, Props, State và các Hooks cơ bản (useState, useEffect).Tuần 15 - 16: Quản lý Router (Chuyển trang không cần tải lại web) và học thêm Tailwind CSS để viết giao diện nhanh hơn.Tuần 17 - 20: Làm dự án tổng hợp (Đăng ký/Đăng nhập, Giỏ hàng, Tìm kiếm, Lọc sản phẩm).Thực hành: Xây dựng một trang Web thương mại điện tử mini hoặc một trang Clone của Netflix/Spotify.
